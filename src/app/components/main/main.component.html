Portfolio

Esta información se actualizará con proyectos en progreso y ya terminados.

RCH Hotel

Proyecto grupal realizado junto a mis compañeros de carrera Ernesto Plata y Carlos Maroto. Se trata de una aplicación Android desarrollado en Java dentro de Android Studio, y consiste en una app orientada a la reserva de habitaciones en una marca de hoteles ficticia.

Super 8 Film

Proyecto grupal realizado junto a mis compañeros de carrera Ernesto Plata y Carlos Maroto. Trata de una aplicación en UWP que simula a Netflix, con distintas secciones para la clasificación y reproducción de vídeos.

Tamagochi

Proyecto personal desarrollado en WPF. El objetivo es crear un juego en el que un dibujo animado que simula ser un Tamagochi debe ser cuidado activamente para mantenerse con vida y conseguir con el paso del tiempo una mejor puntuación.

Premier Tweet

Proyecto grupal realizado junto a mis compañeros de carrera Juán Muñoz, Guadalupe Plaza, Carlos Mohedano y Carlos Maroto. Desarrollado en Python, se utiliza la API de Twitter y de IMDB para recoger Tweets de los últimos estrenos cinematrográficos y aplicar Machine Learning sobre ellos, captando cuales han tenido mejor acogida.
